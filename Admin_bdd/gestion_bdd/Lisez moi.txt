COMMENT FAIRE EN LOCAL :
   Partagez le dossier AxS sous le nom que vous voulez (alias)
   Mettez éventuellement vos BD dans le dossier AxS/try/database
   Tapez http://localhost/alias/index.html dans votre navigateur
COMMENT FAIRE EN LIGNE :
   Mettez "blank_97.mdb" ou "blank_2K.mdb" du dossier "bdd" dans votre dossier database
   Modifiez le fichier "_params.asp" avec vos paramètres
   Exécutez la page "index.html"
