Overview

The reports page is used to generate various reports for a specified reporting period. To generate a report, select a report period using the date selection tools provided on the left hand side, then select a report.

After a report is produced, use the icons at the top of the report to export the report or preview the report in print preview mode. (Icons shown on the right.)

   

[TOP]
Summary

The summary shows summary data based on the dates you select.

  • Page Views - The total number of web requests between the selected reporting period.


  • Visits - The total number of visits between the selected reporting period. A visit is calculated based on the web server's session duration setting (usually 20 minutes). If a web request exceeds the session duration after their last web request, this will be counted as a new visit.


  • Unique Visitors - The total number of unique visitors between the selected reporting period. An unique visitor is calculated based on the IP Address of the visitor.


  • Users Online - The number of unique visitors who have accessed the site within a certain period of time; this is 60 minutes by default.


  • Searches - This displays the number of searches from registered search engines.


  • Avg. Page Views Per Visit - The average number of web requests for each visitor between the selected reporting period.


  • Avg. Page Views Per Hour - The average number of web requests that occur each hour between the selected reporting period.


  • Avg. Page Views Per Day - The average number of web requests that occur each day between the selected reporting period.


  • Avg. Visits Per Day - The average number of visits that occur each day between the selected reporting period.


  • Avg. Unique Visitors Per Day - The average number of unique visitors between the selected reporting period.


  • Unique Pages Logged - The number of unique pages that have been logged.


  • Unique Urls Logged - The number of unique urls that have been logged.

Who's Online

This report shows all of the active visitors on the web site including their IP address, their hostname (if available), the country of origin, their last page access and the total number of page views during their visit.

[TOP]
Pages

This report shows the total number of web requests for each page for the selected reporting period.

[TOP]
Urls

This report shows the total number of web requests for each url including the querystring for the selected reporting period.

[TOP]
Daily

This report shows the total number of web requests for each day for the selected reporting period.

[TOP]
Hourly

This report shows the total number of web requests for each hour for the selected reporting period.

[TOP]
By IP Address

This report shows the total number of page views for each IP Address for the selected reporting period. This report will also display the hostname associated with the IP Address if that data is available.

[TOP]
Directories

This report shows the total number of web requests in each directory or folder for the selected reporting period.

[TOP]
File Types

This report shows the total number of web requests grouped by file type or file extension for the selected reporting period.

[TOP]
Daily Unique Visitors

This report displays the total number of unique visitors for each day for the selected reporting period. An unique visitor is determined by a distinct IP address.

[TOP]
Daily Visits

This report displays the total number of visits each day for the selected reporting period. A visitor is defined as a user session. A new user session occurs when a web request is made by a new visitor that hasn't made a web request within the period defined by the web server's session duration setting.

[TOP]
Visits By IP Address

This report displays the total number of visits for each IP Address for the selected reporting period. This report will also display the hostname of the visitor if that data is available. A visitor is defined as a user session. A new user session occurs when a web request is made by a new visitor that hasn't made a web request within a period defined by the web server's session duration setting.

[TOP]
Entry Pages

This report shows the top entry pages viewed for the selected reporting period. An entry page is the first page view of a visit.

[TOP]
Exit Pages

This report shows the top exit page viewed for the selected reporting period. An exit page is the last page view of a visit.

[TOP]
Domains

This report shows which domains have sent visitors to your site for the selected reporting period.

[TOP]
Hosts

This report shows which host names have sent visitors to your site for the selected reporting period.

[TOP]
Pages

This report shows which specific pages from other web sites have sent visitors to your web site during the specified report period. Querystring data has been removed from this report. If the URL is too long, a .. will replace folder names in the middle of the url (if the truncate urls configuration option is enabled). Urls that have been truncated will have this image after the url . You can mouse over the icon to see the full URL.

[TOP]
Urls

This report shows which URLs (Uniform Resource Locators) have sent visitors to your web site during the specified report period. If the URL is too long, a .. will replace folder names in the middle of the url (if the truncate urls configuration option is enabled). Urls that have been truncated will have this image after the url . You can mouse over the icon to see the full URL.

[TOP]
Extensions

This report shows which domain extensions (TLD - Top Level Domains) have sent visitors to your web site during the specified report period. Examples of domain extensions are .com, .net, .org, .co.uk, etc.

[TOP]
Searches

This report shows the number of searches performed by popular search engines for the reporting period selected.

[TOP]
Keywords

This report shows what keywords are being used to find your site in the search engines for the reporting period selected.

[TOP]
Robots

This report shows what known robots or spiders are accessing the site for the selected reporting period. A robot is a program that automatically traverses the web by retrieving a document, and recursively retrieving all documents that are referenced.

[TOP]
Country Of Origin

This report shows what countries visits originate from for the selected reporting period. Country of origin is determined using the IP address and comparing it to FJstats's IP address to country lookup table in the database.

[TOP]
Browsers

This report shows which browsers visitors are using to view the site with for the selected reporting period.

[TOP]
Operating Systems

This report shows what operating systems visitors were using for the selected reporting period.

[TOP]
Languages

This report shows the visitors language for the selected reporting period. Language is defined by a visitor's browser setting.

[TOP]
Screen Area

This report shows the display resolution of visitors that are accessing the site for the selected reporting period.

[TOP]
User Agents

This report shows all user agents of any web request to the site for the selected reporting period. It will also show what type of browser or robot this was detected as.
